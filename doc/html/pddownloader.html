    Detailed Description

    This is main class responsible for downloading photos from camera/ device.

    PDDownloader does all the heavy lifting when downloading photos.

    All the files matching filter lists (set up in PDSettingsDialog) are copied into a specified folder hierarchy (currently it's <year>/<month>/<day>/) using one of folder handlers PDFolderHandlers
